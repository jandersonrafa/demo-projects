## 1. Java MVC Virtual Threads
**Nomad Gateway Port:** 8101

### Create Bonus
```bash

  -H "Content-Type: application/json" \
  -d '{
    "amount": 500.00,
    "description": "MVC VT Bonus",
    "clientId": "user-789"
  }'
```
